Matrix-Breakout: 2 Morpheus — VulnHub Walkthrough

Platform: VulnHub
Difficulty: Intermediate
Tools Used: nmap, ffuf, LinPEAS, Dirty Pipe exploit, Netcat, Burp Suite, php://filter

Reconnaissance

  • Identified target IP using arp-scan
  • Ran full port scan with nmap revealing ports 22, 80, and 81
  • Apache served Matrix-themed hints

Enumeration

  • Discovered hidden files with ffuf: graffiti.php, robots.txt
  • Exploited graffiti.php for command injection

Exploitation

  • Established reverse shell using Netcat
  • Used LinPEAS to find Dirty Pipe vulnerability
  • Compiled and executed exploit to gain root

Root Flag

Successfully escalated privileges and captured the root flag.

Lessons Learned

  • Thorough enumeration reveals multiple attack surfaces
  • Command injection and php://filter are powerful vectors
  • Dirty Pipe is a real-world privilege escalation exploit worth mastering

Mercury — VulnHub Walkthrough

Platform: VulnHub
Difficulty: Easy
Tools Used: netdiscover, nmap, gobuster, sqlmap, SSH, sudo, vi exploit

Reconnaissance

  • Used netdiscover to identify target IP
  • Scanned with nmap to find open ports: 22 (SSH), 8080 (HTTP)

Enumeration

  • Used gobuster to find hidden directories
  • Found SQL injection vulnerability in mercury facts directory

Exploitation

  • Used sqlmap to dump credentials
  • Logged in via SSH as webmaster
  • Escalated privileges using sudo vi exploit

Root Flag

Gained root access and captured both user and root flags.

Lessons Learned

  • SQL injection can be a powerful entry point
  • Always check sudo -l for privilege escalation paths
  • Simple misconfigurations can lead to full compromise
